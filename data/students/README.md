# Student Photos Directory

This directory is excluded from git for privacy reasons.

## Setup Instructions

1. Create a folder for each student following this naming convention:
   ```
   SXXX_Firstname_Lastname
   ```
   Example: `S000_John_Doe`, `S001_Jane_Smith`

2. Place 3-5 clear face photos of each student in their folder.

3. Run the enrollment script from the project root:
   ```bash
   python -m ml.enroll_batch
   ```

## Getting the Team Dataset

Contact the team lead to receive the student photo dataset via:
- Google Drive (shared link)
- USB transfer
- Any private channel

> **Note:** Do NOT commit student photos to the public repository.
